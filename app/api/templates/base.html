Backend:

Backend was created with django-REST (Python) It utilizes combined powers of PostgreSQL and Redis cache for improved performance. Requirments of the project can be found here.

Frontend:

Front end was designed with simplicity and code independence in mind, requiring zero dependencies to run. This ensures the longevity and best possible performance of the app. Code optimization and packaging was achieved using WebPack module bundler.
